Business Card Data Extraction with OCR and NER

The main objective of this app is to extract entities from the scanned documents like invoice, Business Card, Shipping Bill, Bill of Lading documents etc. However, for the sake of data privacy I have restricted my view to Business Card. But the framework can be utilized to extract data from all kinds of financial documents. To develop this project two main technologies in data science were used

  1. Computer Vision
  2. Natural Language Processing

In Computer Vision module, the document is scanned to identify the location of text and text is extracted from the image. Then in Natural language processing, the entitles from the text are extracted and necessary text cleaning is done and the entities are parsed from the text.

Python Libraries used in Computer Vision Module.

  • OpenCV
  • Numpy
  • Pytesseract

Python Libraries used in Natural Language Processing

  • SpaCy
  • Pandas
  • Regular Expression
  • String
